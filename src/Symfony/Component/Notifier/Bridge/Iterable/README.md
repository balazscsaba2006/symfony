# Iterable Notifier

Provides [Iterable](https://iterable.com/) integration for Symfony Notifier.

## DSN example

```
ITERABLE_DSN=iterable://API_KEY@default?campaign_id=ID
```

where:

-   `API_KEY` is your Iterable API key
